LP2DPBO2024C2
Pada Latihan Praktikum 2 DPBO Mengenai Multi-Level Inheritance, yang dimana pada Class Product merupakan suatu orang tua dari Class Clothing serta Class Clothing merupakan Orang Tua dari Class Shirt.
